
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman game that reinforces Python skills in strings, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Game Setup and Word Selection

#### Description

Create the game setup so it can choose a random word and display the puzzle state to the player.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Display the hidden word using underscores for unguessed letters
- Update and show the current puzzle state after each guess

### 🛠️ Gameplay Logic and Win/Loss Conditions

#### Description

Implement the main game loop, letter guessing, and end conditions for winning or losing.

#### Requirements
Completed program should:

- Accept letter guesses from the player
- Track and display remaining incorrect attempts
- End the game when the player guesses the word or runs out of attempts
- Show a win message when the word is guessed and a lose message when attempts are exhausted
